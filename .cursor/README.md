Epigenetic IDE: **Archivist memory**, **Command Deck buttons**, **Inspiration Forge**, **Golden Master MCP subagents**.

> No slash commands. Use the extension sidebar.

## Install extension

`.\scripts\install-omni-viewport.ps1` (Windows) → Reload Window → activity bar **Omni-Viewport**

## Setup

1. `pip install mcp` (or `pip install -r requirements.txt`)
2. `Ctrl+Shift+J` → enable all `utah_*` and `zeo_*` servers from `.cursor/mcp.json`
3. **Developer: Reload Window**
4. Activity bar → **Omni-Viewport**

## Personal settings (do not commit)

- `~/.cursor/mcp.json` — global MCP overrides
- `.cursor/user.json` — local cursor overrides (gitignored)
- `utahOmniViewport.inspirationRoots` — workspace settings only
