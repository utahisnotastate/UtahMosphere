# IDE epigenética de Cursor (nivel 6)

UtahMosphere configura Cursor como **Omni-Viewport** — no una máquina de escribir. El IDE hereda el vibe del proyecto vía memoria epigenética, **botones del Command Deck** y **subagentes MCP autónomos**.

> **Sin comandos slash.** Instala la [extensión Utah Omni-Viewport](OMNI_VIEWPORT_EXTENSION.md) y usa los botones de la barra lateral.

## Utah-Archivist (`.cursor/memory.md`)

Memoria subconsciente autoevolutiva. El agente lee antes de cada acción y agrega patrones de forma autónoma.
