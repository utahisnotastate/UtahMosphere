# Omni-Desk — tienda de apps soberana (v35.0)

**Omni-Desk** es el escritorio holográfico acelerado por GPU de UtahMosphere. En Time-3 no descargamos apps — **las manifestamos**.

## Suite Genesis

| ID | Nombre | Propósito |
|----|--------|-----------|
| `web_forge` | **Omni-WebForge** | Sitio desde doc, repo o voz |
| `zeo_canvas` | **ZEO-Canvas** | Generador de imágenes local |
| `app_smith` | **Kinematic-AppSmith** | UI descrita → cliente Ghost Tune |
| `holo_notebook` | **Holographic-Notebook** | PDF/código → herramientas autónomas |
| `claw_harvester` | **UtahClaw-Harvester** | Extracción MCP de codebases |
