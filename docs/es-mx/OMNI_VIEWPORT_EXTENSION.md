# Extensión Utah Omni-Viewport

**Utah Omni-Viewport** reemplaza comandos slash con **paneles GUI clicables** para flujos Level 6.

Fuente: `extensions/utah-omni-viewport/`

## Instalación

`Ctrl+Shift+P` → **Developer: Install Extension from Location** → `extensions/utah-omni-viewport`

## Paneles

| Panel | Propósito |
|-------|-----------|
| Command Deck | Protocolos Level 6 con un clic |
| Inspiration Forge | Escaneo multi-repo + plan de función |

Rutas personales (`inspirationRoots`) — solo workspace, no en git.

Ver [guía en inglés](../OMNI_VIEWPORT_EXTENSION.md).
