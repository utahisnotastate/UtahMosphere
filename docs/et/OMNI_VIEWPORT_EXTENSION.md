# Utah Omni-Viewporti laiendus

**Utah Omni-Viewport** asendab lõikude käsud **klikitavate GUI paneelidega** Level-6 UtahMosphere töövoogude jaoks.

Allikas: `extensions/utah-omni-viewport/`

## Paigaldamine

1. `Ctrl+Shift+P` → **Developer: Install Extension from Location**
2. Vali `extensions/utah-omni-viewport`
3. Laadi aken uuesti.

## Paneelid

| Paneel | Eesmärk |
|--------|---------|
| **Command Deck** | Level-6 protokollid ühe klõpsuga |
| **Inspiration Forge** | Mitme koodibaasi skaneerimine + funktsiooniplaneerimine |

## Inspiration Forge

1. Üks kaust rida kohta.
2. Valikuline funktsiooni vihje.
3. **Scan & Plan** — `scripts/inspiration_scanner.py`
4. **Send Plan to Composer** — `[PROTOCOL: INSPIRATION-PLAN]`

Isiklikud teed — ainult tööruumi seadetes, mitte gitis.

Vaata [inglise juhendit](../OMNI_VIEWPORT_EXTENSION.md).
