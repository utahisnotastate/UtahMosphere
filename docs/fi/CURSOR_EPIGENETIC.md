# Cursorin epigeneettinen IDE (taso 6)

UtahMosphere määrittää Cursorin **Omni-Viewportiksi** — ei kirjoituskoneeksi. IDE perii projektin tunnelman epigeneettisestä muistista, **Command Deck -painikkeista** ja **autonomisista MCP-aliedustajista**.

> **Ei slash-komentoja.** Asenna [Utah Omni-Viewport -laajennus](OMNI_VIEWPORT_EXTENSION.md) ja käytä sivupalkin painikkeita.

## Utah-Archivist (`.cursor/memory.md`)

Itseään päivittävä alitajunnan muisti. Agentti lukee ennen jokaista toimintoa ja lisää kuvioita autonomisesti.

## Inspiration Forge

Suunnittele ominaisuuksia kaivamalla kuvioita **valitsemistasi koodipohjista**. Henkilökohtaiset polut — vain workspace-asetuksissa.
