# Utah Omni-Viewport -laajennus

**Utah Omni-Viewport** korvaa slash-komennot **napsautettavilla GUI-paneeleilla** Level 6 -työnkuluille.

Lähdekoodi: `extensions/utah-omni-viewport/`

## Asennus

`Ctrl+Shift+P` → **Developer: Install Extension from Location** → `extensions/utah-omni-viewport`

## Paneelit

| Paneeli | Tarkoitus |
|---------|-----------|
| Command Deck | Level 6 -protokollat yhdellä napsautuksella |
| Inspiration Forge | Usean repojen skannaus + ominaisuussuunnitelma |

Henkilökohtaiset polut (`inspirationRoots`) — vain workspace, ei gitiin.

Katso [englanninkielinen opas](../OMNI_VIEWPORT_EXTENSION.md).
