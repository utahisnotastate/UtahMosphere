# IDE épigénétique Cursor (niveau 6)

UtahMosphere configure Cursor comme **Omni-Viewport** — pas une machine à écrire. L'IDE hérite du « vibe » du projet via la mémoire épigénétique, les **boutons Command Deck** et des **sous-agents MCP autonomes**.

> **Pas de commandes slash.** Installez [l'extension Utah Omni-Viewport](OMNI_VIEWPORT_EXTENSION.md) et utilisez la barre latérale.

## Utah-Archivist (`.cursor/memory.md`)

Mémoire subconsciente auto-évolutive. L'agent lit avant chaque action et ajoute des motifs sans demander.

## Command Deck

| Bouton | Protocole | Action |
|--------|-----------|--------|
| Cascade Sync | `CASCADE` | Sync holographique sur tout le stack |
| Absorb | `ABSORB` | Assimiler du code externe |
| Void Fill | `VOID-FILL` | Recherche UtahClaw MCP uniquement |
| Chrono Fix | `CHRONO-FIX` | Débogage par retour arrière terminal |
| Entropy Purge | `ENTROPY-PURGE` | Réduire les 3 points d'entropie |
| Schism | `SCHISM` | Extraction microservice + ADR |
| Vibe Shift | `VIBE-SHIFT` | Restyle UI seulement |
| Immortalize | `IMMORTALIZE` | Tests + ADR + commande commit |
| Evolve Folder | `EVOLVE-FOLDER` | Évolution autonome du dossier |

## Inspiration Forge

Planifiez des fonctionnalités en minant des motifs dans **vos dépôts sélectionnés**. Chemins personnels — paramètres workspace uniquement.
