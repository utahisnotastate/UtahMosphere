# Extension Utah Omni-Viewport

**Utah Omni-Viewport** remplace les commandes slash par des **panneaux GUI cliquables** pour les flux Level 6.

Source : `extensions/utah-omni-viewport/`

## Installation

`Ctrl+Shift+P` → **Developer: Install Extension from Location** → `extensions/utah-omni-viewport`

## Panneaux

| Panneau | Rôle |
|---------|------|
| Command Deck | Protocoles Level 6 en un clic |
| Inspiration Forge | Scan multi-dépôts + plan de fonctionnalité |

Chemins personnels (`inspirationRoots`) — workspace seulement, pas dans git.

Voir [guide anglais](../OMNI_VIEWPORT_EXTENSION.md).
