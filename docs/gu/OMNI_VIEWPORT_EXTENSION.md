# Utah Omni-Viewport Extension

**Utah Omni-Viewport** replace slash commands yan **clickable GUI panels** para Level 6 workflows.

Source: `extensions/utah-omni-viewport/`

## Install

`Ctrl+Shift+P` → **Developer: Install Extension from Location** → `extensions/utah-omni-viewport`

## Panels

| Panel | Purpose |
|-------|---------|
| Command Deck | Level-6 protocols one click |
| Inspiration Forge | Multi-repo scan + feature plan |

Personal paths (`inspirationRoots`) — workspace only, not in git.

See [English guide](../OMNI_VIEWPORT_EXTENSION.md).
