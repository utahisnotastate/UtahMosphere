# Эпигенетическая IDE Cursor (уровень 6)

UtahMosphere настраивает Cursor как **Omni-Viewport** — не печатную машинку. IDE наследует «вайб» проекта через эпигенетическую память, **кнопки Command Deck** и **автономных MCP-субагентов**.

> **Без slash-команд.** Установите [расширение Utah Omni-Viewport](OMNI_VIEWPORT_EXTENSION.md) и нажимайте кнопки на боковой панели.

## Utah-Archivist (`.cursor/memory.md`)

Самопереписывающаяся подсознательная память. Агент читает перед каждым действием, дописывает паттерны автономно.

## Command Deck

| Кнопка | Протокол | Действие |
|--------|----------|----------|
| Cascade Sync | `CASCADE` | Голографическая синхронизация всего стека |
| Absorb | `ABSORB` | Ассимиляция чужого кода |
| Void Fill | `VOID-FILL` | Исследование через UtahClaw MCP |
| Chrono Fix | `CHRONO-FIX` | Откат по последней ошибке терминала |
| Entropy Purge | `ENTROPY-PURGE` | Схлопывание топ-3 точек энтропии |
| Schism | `SCHISM` | Выделение микросервиса + ADR |
| Vibe Shift | `VIBE-SHIFT` | Только рестайл UI |
| Immortalize | `IMMORTALIZE` | Тесты + ADR + команда commit |
| Evolve Folder | `EVOLVE-FOLDER` | Автономная эволюция папки |

## Inspiration Forge

Планируйте фичи, анализируя **выбранные вами кодовые базы**. Личные пути — только в настройках workspace.
