# Omni-Glass UI (v34.0)

**Omni-Glass** заменяет логи терминала и Datadog визуальным многообразием активности агентов — векторы мыслей Omni-Mind, пути исследования UtahClaw и мутации Lazarus AST.

## Многообразие состояния

См. `GET /omni/glass` на порту 8999 и SSE `GET /stream` на порту 9091.

## Переменные окружения

| Переменная | По умолчанию | Назначение |
|------------|--------------|------------|
| `UTAH_OMNI_GLASS_STREAM` | `1` | SSE relay на 9091 |
| `UTAH_OMNI_GLASS_PORT` | `9091` | Порт FluxRelay |
