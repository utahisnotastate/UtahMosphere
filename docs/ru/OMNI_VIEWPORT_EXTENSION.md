# Расширение Utah Omni-Viewport

**Utah Omni-Viewport** заменяет slash-команды **кнопками GUI** для рабочих процессов UtahMosphere Level 6.

Исходники: `extensions/utah-omni-viewport/`

## Установка

`Ctrl+Shift+P` → **Developer: Install Extension from Location** → `extensions/utah-omni-viewport`

## Панели

| Панель | Назначение |
|--------|------------|
| Command Deck | Протоколы Level 6 одним кликом |
| Inspiration Forge | Сканирование нескольких репозиториев + план фичи |

## Inspiration Forge

1. Одна папка на строку.
2. Подсказка фичи (необязательно).
3. **Scan & Plan** — `scripts/inspiration_scanner.py`
4. **Send Plan to Composer** — `[PROTOCOL: INSPIRATION-PLAN]`

Личные пути (`inspirationRoots`) — только workspace, не в git.

См. [английский гид](../OMNI_VIEWPORT_EXTENSION.md).
