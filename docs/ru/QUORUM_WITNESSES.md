# Кворум-свидетели в нескольких регионах (v32.0)

**Узлы-свидетели** — лёгкие региональные наблюдатели, хранящие криптографические хеши консенсуса роя. Если регион (например, US-East) теряет магистральную связь, свидетели в **Океании**, **Европе** и **Азии** выступают арбитрами — сохраняя целостность роя без централизованного управления.

## Архитектура

```
US-East Node (partitioned)     Witness Layer
        |                           |
        |-- state_hash ------------>|-- us-east witness
        |                           |-- eu-west witness
        |                           |-- oceania-apac witness
        |                           |-- asia-east witness
        |<-- quorum confirmed ------|   (>51% must agree)
```

## Модуль (`quorum_witness.py`)

| Метод | Назначение |
|-------|------------|
| `get_consensus(proposed_state_hash)` | >50% свидетелей должны подтвердить хеш |
| `ping_and_verify(hash)` | HTTP-пинг свидетеля по региону |
| `record_local_witness(hash)` | Локальный арбитр при недоступности удалённых |
| `export_witnesses()` | Статус региональных свидетелей |

Регионы по умолчанию: `us-east`, `eu-west`, `oceania-apac`, `asia-east`

## HTTP API

### GET /witness/status

```bash
curl http://127.0.0.1:8999/witness/status
```

## Переменные окружения

| Переменная | По умолчанию | Назначение |
|------------|--------------|------------|
| `UTAH_WITNESS_ENFORCE` | `1` | Требовать кворум свидетелей (`0` = разработка) |
| `UTAH_WITNESS_THRESHOLD` | `0.51` | Минимальная доля голосов |
| `UTAH_WITNESS_NODES` | 4 по умолчанию | Конечные точки свидетелей через запятую |
