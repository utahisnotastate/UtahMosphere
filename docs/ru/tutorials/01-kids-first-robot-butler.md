# Учебник: Ваш первый робот-дворецкий

**Аудитория:** Дети и семьи  
**Время:** 15 минут  
**Понадобится:** Компьютер с установленным UtahMosphere, микрофон (опционально)

---

## Что вы создадите

Крошечного «Робота-дворецкого» на компьютере, который слушает вас и развёртывает приложения по просьбе.

---

## Шаг 1: Знакомство с дворецким

UtahMosphere — как робот-дворецкий в маленькой коробке (мини-ПК или Raspberry Pi). Вместо оплаты большой компании за хостинг ваших вещей дворецкий живёт в **вашей** комнате.

Запустите «мозг» дворецкого:

```bash
python utahmosphere_os.py
```

Попросите взрослого помочь установить `UTAH_DATA_DIR` в папку на компьютере, если вы не на Linux.

---

## Шаг 2: Поздоровайтесь с дворецким

Откройте второе окно и выполните:

```bash
python voice_bridge.py
```

Когда появится **"Listening..."**, попробуйте сказать:

> **"Claim node"**

Это учит дворецкого вашему голосу. Как ключ, которым может пользоваться только ваш голос.

---

## Шаг 3: Постройте лимонадный киоск

Скажите:

> **"Deploy application lemonade"**

Дворецкий создаёт крошечное приложение «лимонадный киоск» на компьютере. Проверьте, что всё сработало:

```bash
curl http://127.0.0.1:8999/status
```

Найдите `"lemonade"` в списке арендаторов.

---

## Шаг 4: Нет микрофона? Не проблема!

Попросите взрослого выполнить вместо этого:

```bash
python examples/voice-deploy-simulator/deploy.py lemonade
```

Тот же результат — дворецкий всё равно построит киоск.

---

## Шаг 5: Смотрите на панель управления

Если есть экран, запустите:

```bash
python flux_gui.py
```

Вы увидите зелёный текст с тем, что делает дворецкий — как приборная панель космического корабля!

---

## Чему вы научились

- **Claim node** = научить дворецкого вашему голосу
- **Deploy application** = построить что-то новое
- Дворецкий ведёт список всего, что построил

---

## Весёлые задания

1. Разверните три приложения: `toys`, `games` и `art`
2. Скажите **"status grid"** и прочитайте отчёт дворецкого
3. Нарисуйте Робота-дворецкого и подпишите: Голос, Мозг, Приложения
