# Учебник: Настройка без жаргона

**Аудитория:** Нетехнические пользователи, владельцы малого бизнеса  
**Время:** 20 минут (с помощником, разбирающимся в технике)  
**Цель:** Запустить UtahMosphere и развернуть первое приложение

---

## Что такое UtahMosphere?

Представьте **небольшой компьютерный мозг**, который запускает ваш сайт или приложение **в офисе или дома** — без ежемесячного облачного счёта от Amazon или Google.

Можно даже **говорить с ним**: «Deploy application my-store» — и он всё настроит.

Полное руководство простым языком: [Руководство без жаргона](../NON_TECHNICAL_GUIDE.md)

---

## Что вам понадобится

| Что | Зачем |
|-----|-------|
| Мини-ПК или Raspberry Pi | «Мозг» — оборудование |
| Интернет (для установки) | Однократная загрузка ПО |
| Помощник по технике (опционально) | Для команды установки |
| USB-микрофон (опционально) | Для голосового управления |

---

## Шаг 1: Установка «мозга»

Помощник выполняет **одну команду** на мини-ПК (Linux):

```bash
sudo bash setup.sh
```

Всё устанавливается автоматически. Около 10–15 минут.

**Нет Linux?** Помощник может использовать Docker:

```bash
docker-compose up -d
```

---

## Шаг 2: Проверка работы

Помощник открывает браузер или терминал и проверяет:

```bash
curl http://127.0.0.1:8999/health
```

Если видите `"healthy"` — мозг проснулся.

---

## Шаг 3: Обучение вашему голосу (опционально)

Помощник запускает:

```bash
python voice_bridge.py
```

Скажите чётко: **"Claim node"**

Теперь только ваш голос (или одобренный помощник) управляет системой.

---

## Шаг 4: Публикация приложения

**Голосом:** скажите **"Deploy application my-store"**

**Без голоса:** помощник выполняет:

```bash
python examples/voice-deploy-simulator/deploy.py my-store
```

Всё. Никаких сложных настроек сервера.

---

## Шаг 5: Что сейчас работает

Помощник может открыть зелёную панель:

```bash
python flux_gui.py
```

Или проверить с любого компьютера в той же сети:

```bash
curl http://YOUR-MINI-PC-IP:8999/status
```

## Повседневные задачи (спросите помощника)

| Вы хотите… | Скажите или попросите… |
|------------|------------------------|
| Добавить приложение | "Deploy application [name]" |
| Проверить, что работает | "Status grid" |
| Открыть панель | Запустить Utah-Flux GUI |
| Исправить проблему | Перезапуск: `sudo systemctl restart utahmosphere` |

---

## Обещание «без обслуживания»

UtahMosphere сам очищает старые ресурсы и восстанавливается в фоне. Настроили один раз — и работает.

---

## Глоссарий

| Слово | Простое значение |
|-------|------------------|
| **Deploy** | Разместить приложение на «мозге» |
| **Claim node** | Научить «мозг» вашему голосу |
| **Tenant** | Одно приложение в системе |
| **Healthy** | «Мозг» работает нормально |
