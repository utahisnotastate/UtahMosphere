# Cursor epigenetiskt IDE (nivå 6)

UtahMosphere konfigurerar Cursor som **Omni-Viewport** — inte en skrivmaskin. IDE:t ärver projektets vibe via epigenetiskt minne, **Command Deck-knappar** och **autonoma MCP-underagenter**.

> **Inga slash-kommandon.** Installera [Utah Omni-Viewport-tillägget](OMNI_VIEWPORT_EXTENSION.md) och använd sidofältets knappar.

## Utah-Archivist (`.cursor/memory.md`)

Självskrivande undermedvetet minne. Agenten läser före varje åtgärd och lägger till mönster autonomt.
