# Utah Omni-Viewport-tillägg

**Utah Omni-Viewport** ersätter slash-kommandon med **klickbara GUI-paneler** för Level 6-arbetsflöden.

Källa: `extensions/utah-omni-viewport/`

## Installation

`Ctrl+Shift+P` → **Developer: Install Extension from Location** → `extensions/utah-omni-viewport`

## Paneler

| Panel | Syfte |
|-------|-------|
| Command Deck | Level 6-protokoll med ett klick |
| Inspiration Forge | Flera repon + funktionsplan |

Personliga sökvägar (`inspirationRoots`) — workspace endast, inte i git.

Se [engelsk guide](../OMNI_VIEWPORT_EXTENSION.md).
