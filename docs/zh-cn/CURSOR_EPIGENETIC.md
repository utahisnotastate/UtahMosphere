# Cursor 表观遗传 IDE（Level 6）

UtahMosphere 将 Cursor 配置为 **Omni-Viewport**——不是打字机。IDE 通过表观遗传记忆、**Command Deck 按钮**和**自主 MCP 子代理**继承项目 vibe。

> **无 slash 命令。** 安装 [Utah Omni-Viewport 扩展](OMNI_VIEWPORT_EXTENSION.md)，使用活动栏按钮。

## Utah-Archivist（`.cursor/memory.md`）

自我重写的潜意识记忆。代理在每次操作前读取，自主追加模式，无需请示。

## Command Deck

| 按钮 | 协议 | 动作 |
|------|------|------|
| Cascade Sync | `CASCADE` | 全栈全息同步 |
| Absorb | `ABSORB` | 同化外部代码 |
| Void Fill | `VOID-FILL` | 仅 UtahClaw MCP 调研 |
| Chrono Fix | `CHRONO-FIX` | 终端错误时间回溯调试 |
| Entropy Purge | `ENTROPY-PURGE` | 坍缩前三熵热点 |
| Schism | `SCHISM` | 微服务拆分 + ADR |
| Vibe Shift | `VIBE-SHIFT` | 仅 UI 换肤 |
| Immortalize | `IMMORTALIZE` | 测试 + ADR + commit 命令 |
| Evolve Folder | `EVOLVE-FOLDER` | 自主演进文件夹 |

## Inspiration Forge

从**你选择的代码库**挖掘模式以规划新功能。个人路径仅保存在工作区设置中。
