# Omni-Desk 主权应用商店 (v35.0)

**Omni-Desk** 是 UtahMosphere 的 GPU 加速全息桌面。在 Time-3 中我们不下载应用——**我们显化它们**。

## Genesis 套件

| ID | 名称 | 用途 |
|----|------|------|
| `web_forge` | **Omni-WebForge** | 从文档、仓库或语音生成网站 |
| `zeo_canvas` | **ZEO-Canvas** | 本地图像生成 |
| `app_smith` | **Kinematic-AppSmith** | 描述 UI → Ghost Tune 客户端 |
| `holo_notebook` | **Holographic-Notebook** | PDF/代码 → 自主工具 |
| `claw_harvester` | **UtahClaw-Harvester** | 从代码库提取 MCP 工具 |

## HTTP API（内核 8999）

- `GET /desk/apps`
- `POST /desk/intent`
