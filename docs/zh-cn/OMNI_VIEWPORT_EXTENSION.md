# Utah Omni-Viewport 扩展

**Utah Omni-Viewport** 用**可点击 GUI 面板**替代 slash 命令，用于 Level 6 工作流。

源码：`extensions/utah-omni-viewport/`

## 安装

`Ctrl+Shift+P` → **Developer: Install Extension from Location** → `extensions/utah-omni-viewport`

## 面板

| 面板 | 用途 |
|------|------|
| Command Deck | 一键 Level 6 协议 |
| Inspiration Forge | 多仓库扫描 + 功能规划 |

个人路径（`inspirationRoots`）——仅工作区设置，勿提交 git。

参见[英文指南](../OMNI_VIEWPORT_EXTENSION.md)。
