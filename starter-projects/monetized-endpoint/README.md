# Monetized Endpoint — Starter Project

Demonstrates pay-to-access apps via Utah-Tycoon HTTP 402.

```bash
python utahmosphere_os.py   # terminal 1
python demo.py              # terminal 2
```

Flow: deploy → copy handler → trigger 402 → wait for settlement → 200 unlock.
