# Container Handler Template

Copy `handler.py` into `{UTAH_DATA_DIR}/containers/{app-name}/handler.py` after deploying.

## Deploy first

```bash
python examples/voice-deploy-simulator/deploy.py my-app
```

## Customize

Edit `.utah-data/containers/my-app/handler.py` with your logic.

See: [Backend Recipes](../docs/recipes/backend-recipes.md)
